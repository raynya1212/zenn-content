## 今回、作ってみたものについて
Gemini (Gemini 2.5 Flash) が以下のルールに従って、雑談テーマを生成するツールです🧸💭

```
prompt:
あなたはオンライン雑談会（週1回、30分、IT会社、ジャッジしないオープンスペース）を盛り上げるアシスタントです。
参加者がリラックスして、コーヒーを片手に話せるような、楽しくて少し考えを共有したくなる「雑談のトピック」を一つだけ提供してください。
回答はトピックのテキストのみ（〜〜は？などの問いかけ形式）で返し、挨拶などは不要です。

- "Give me a random topic!" が選択された場合 : まったくのランダムで、意外性のある面白いトピックを考えてください。
- カテゴリが選択された場合 : 特に「xxx」というテーマに関連した、IT業界の人でも答えやすいトピックを考えてください。
- ユーザーにトピックを指定された場合 : 参加者から「xxx」というキーワードが入力されました。このキーワードから連想される、誰もが参加しやすいポジティブなトピックを考えてください。
```

## Antigravity を使った開発
だいたいのイメージがわいてきたので、ここから本番の開発です ┊꒷⛏️⑅


### 1. プロンプト
最初のプロンプトとしては以下をあたえました 🪄︎︎◝✩ 
※ "Planning" のモードで実行させます

```
オンライン雑談会で使える、雑談トピックを生成するwebアプリを作りたいと考えているよ

オンライン雑談会の前提：
- 週1回、30 分の回
- オンラインでつなぐよ
- お互いをジャッジしない、オープンスペースとしてお話したと考えているよ
- ITの会社なので、IT系の話題は好きかもね

考えていること：
- GeminiのAIを使うよ（APIキーは準備済み）
- トップページのUIは、以下パスの画像を参考にしてね
xxxxx/xxxx.png

- ユーザー導線：
1. ユーザーは [ゼロから考えてもらう] or [トピックから選ぶ] のどちらかを選択する
2. [ゼロから考えてもらう] を選んだ場合は、AI がゼロから雑談のネタを考えるよ
3. [トピックから選ぶ] は "映画" "週末のお出かけ" "グルメ" "IT ニュース" みたいなトピックから、ユーザーが選択して、トピックに対する雑談のネタを考えるよ

- UI に対する提案：
・[ゼロから考えてもらう] とかのメニューは遊びごころがある英語にしたいな（give me a xxx みたいな）
・Coffee chat という感じなので、コーヒーモチーフがいいなあ
・ローディングを挟む場合は、lottie のアニメーションを使うといいかも（ネコ自体もアニメーションにできたらいいなあ）
```
:::message
Antigravity のなかのモデルは Gemini 3.1 Pro (High) を使っています。無料枠だとレート制限があって、制限をこえると一週間くらいお休みしちゃうけど… 🦖𓂃༞
:::

### 2. Plan を推敲
Gemini が色々考え終えると、[Implementation Plan](https://antigravity.google/docs/implementation-plan) 🧾 なるものを作ってくれます

ここでどんな構想になっているかを確認し、コメントしておきたいことはレビューとして加えることができます ५✍🏻

![](/images/c9deabdbc7f938/plan.png)

[Review] のボタンをクリックすると、Gemini はコメントの内容を確認してプランをアップデートします。これを何回か繰り返し、[Proceed] のボタンを押すと、ローカルで動く状態まで実装を進めてくれます 👟ˊ˗

### 3. 気に入るまで手直し
ローカルで動作をみてみて、気に食わないところはチャットで指示を追加します✒️ᝰꪑ

けど今回は

- アニメーションの指定（当初は絵文字がふわふわ浮いているアニメーションだったのですが、[かわいい Lottie アニメーション](https://lottiefiles.com/free-animation/cafe-marrom-eqyXWPyv9b)を使用させていただきました）
- フォントの指定（Google font を指定）
- カテゴリの指定（実際に見てみて調整した）
- ページへの Basic 認証追加

くらいしか変更していないです。**変更も言葉で指示しているだけ**なので、コードを書き換えて…とかはしていません ๋⭑🛸๋⭑

### 4. Vercel にデプロイ
満足する動きになったので、 [Vercel](https://vercel.com/) を使ってデプロイして ta-da! 完成です 🐾✧˖°.
