# ✨AI-900対策ノート：テキスト分析 & NLP まとめ 🧠

2025-04-13 · Reina  
※ 試験対策用のメモ ✍️

## ① Azure Text Analytics の主な機能
| 機能 | 説明 | 例 |
|------|------|------|
| 感情分析 | 文の感情を分類 | 「最高！」→ポジティブ |
| 言語検出 | 文章が何語か判定 | "Hello"→英語 |
| キーフレーズ抽出 | 重要語を抽出 | 「AI が便利」→AI, 便利 |
| 固有表現抽出（NER） | 人名・地名などの固有名詞抽出 | 田中さん＝人名 |

## ② ベクター化とは？
文章を **AI が理解できる数値ベクトルに変換** すること。

例：
```
[0.7, 0.1, 0.5, 0.3]  # 「おはよう」
[0.6, 0.2, 0.4, 0.4]  # 「こんにちは」
```

## ③ 頻度分析 vs N-gram
| 手法 | 説明 | 目的 |
|------|------|------|
| 頻度分析 | 単語の出現回数を数える | よく使う単語を知る |
| N-gram | 単語を連続セットで扱う | 文脈や語の繋がりを見る |

## 📝 試験で出やすいポイント
- 「ベクター化」は超重要ワード
- NLP の各機能はベクター情報を使って判定している
- N-gram は文脈理解のベース
